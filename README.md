# InfoQ podcast

A podcast feed for InfoQ presentations.

## Installation

Either using Leiningen:

    $ lein run

Or using Java:

    $ java -jar infoq-podcast-0.1.0-standalone.jar

## Usage

[http://infoq-podcast.herokuapp.com/](remote) or
[http://localhost:8090/](local).
